# Student Record Management System 

##  Description

This is a simple console-based Java application that allows users to manage student records.
The program is menu-driven and lets users perform basic operations like adding, viewing, searching, and deleting student names.

---

## Features

*  Add a new student
*  View all students
*  Search for a student
*  Delete a student
*  Exit the program

---

## Technologies Used

* Java
* ArrayList (for dynamic data storage)
* Scanner (for user input)

---

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/harinii037/student-record-management-system.git
   ```

2. Navigate to the folder:

   ```bash
   cd StudentManager
   ```

3. Compile the program:

   ```bash
   javac StudentManager.java
   ```

4. Run the program:

   ```bash
   java StudentManager
   ```
